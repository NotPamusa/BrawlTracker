# website design guidelines and recommendations

## 🎨 Palette & Visual Style
 Deep navy base: dark, slightly desaturated blue. Neon-like borders act as the main light source.
Segmented color coding:
Purple/Pink: “Free to Play” and Power Points—vibrant but accessible
Gold/Yellow: “Brawl Pass” and Coins—premium but still attainable
Teal/Green: “Max Spending”—fastest progression, highest value
High-contrast borders: all interactive elements use thick 2–3px borders. This strong outlining improves clarity and separation.

## 📐 Geometry & Shapes
 Octagonal and beveled corners: replaces standard rounded corners. Inputs and cards use chamfered edges and hex-like silhouettes for a sharp, technical feel.
Inset depth: navbar and footer appear recessed using inner shadows, while main cards feel elevated above the background.

## 🖱️ Navigation & Search
 Integrated nav search: subtle “ghost” input in the top bar for quick access.
Hero search bar: central input is the main focal point, highlighted with a soft teal glow to guide attention immediately.
Action buttons: primary CTA (e.g., “Log In”) uses a strong cyan glow for clear visual priority.

## 📊 Content Hierarchy (Impact Cards)
 Card-within-card layout: layered structure to group related data.
Typography: key numbers (e.g., “520 days”) use bold, heavy weights to dominate visual hierarchy.
Iconography: small, recognizable icons placed next to values for quick, glanceable context.

## ✨ Technical Implementation (Tailwind Hints)
 Borders: combine border-t-2 border-l-2 with opacity variation for subtle lighting effect.
Glow: use shadow-[0_0_15px_rgba(20,255,255,0.3)] for soft neon highlights.
Background: radial gradient from slightly lighter blue center to deep navy edges.

## 📝 Tone & Copy
 Despite the high-tech visual style, text should remain clear and natural. Slightly informal tone is fine, but avoid slang or exaggerated phrasing. Clarity and correctness come first.
